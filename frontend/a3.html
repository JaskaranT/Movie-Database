Movie Database API Documentation

Base URL

http://localhost:3000

Endpoints

GET /movies

Retrieve all movies.


GET /movies
Response:
[
    {
        "id": 1,
        "title": "The Godfather",
        "director": "Francis Ford Coppola",
        "release_year": 1972,
        "genre": "Crime",
        "rating": 9.2
    },
    ...
]

GET /movies/:id

Retrieve a movie by its ID.


GET /movies/1
Response:
{
    "id": 1,
    "title": "The Godfather",
    "director": "Francis Ford Coppola",
    "release_year": 1972,
    "genre": "Crime",
    "rating": 9.2
}

POST /movies

Create a new movie.


POST /movies
Request Body:
{
    "title": "The Godfather",
    "director": "Francis Ford Coppola",
    "release_year": 1972,
    "genre": "Crime",
    "rating": 9.2
}
Response:
Movie added with ID: 1

PUT /movies/:id

Update a movie by its ID.


PUT /movies/1
Request Body:
{
    "title": "The Godfather",
    "director": "Francis Ford Coppola",
    "release_year": 1972,
    "genre": "Crime",
    "rating": 9.2
}
Response:
Movie updated with ID: 1

PUT /movies/title/:title

Update a movie rating by its title.


PUT /movies/title/The%20Godfather
Request Body:
{
    "rating": 9.5
}
Response:
Rating updated successfully for movie: The Godfather

DELETE /movies/:id

Delete a movie by its ID.


DELETE /movies/1
Response:
Movie deleted with ID: 1
